# LibraryBookDatabase
Overview
This is a web-based library management system built with CodeIgniter 4 (a PHP framework). It allows users to manage a collection of books by performing basic CRUD (Create, Read, Update, Delete) operations. The application is designed to be simple, user-friendly, and efficient.
Key Features
Book Management: Users can add, view, edit, and delete books.
Cover Image Upload: Each book can have a cover image uploaded and displayed.
Responsive Design: The user interface is built with Bootstrap, ensuring it works well on both desktop and mobile devices.
Validation: Input validation ensures that only valid data is stored in the database.
Technical Stack
Backend: PHP (CodeIgniter 4)
Database: MySQL
Frontend: HTML, CSS (Bootstrap), JavaScript
Server: Apache (via CodeIgniter's built-in development server)
